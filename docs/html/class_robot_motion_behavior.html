Update the position for an ArenaEntity, based on its current position and velocity.

RobotMotionHandler manages the modification to the velocity based on user input and collisions. RobotMotionBehavior translates velocity and position to a new position. Both of these are straightforward, but the framework allows for more sophisticated models of motion in which each wheel has distinct speed.
